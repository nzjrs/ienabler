ienabler by nzjrs

Internet Enabler Clone for University of Canterbury

This is a Linux re-implementation of the University of Canterbury IEnabler client necessary for accessing the Internet from within the UoC network. The package contains two tools, ienabler-gui.py which is a graphical client using Gtk, and ienabler-cmd.py, which is a simple wrapper around the existing university telnet interface.

Dependencies

Python pygtk (optional, for graphical version only) NetworkManager (optional, for graphical version only)

Install

You may install using python-setuptools * python setup.py install Or you may create a deb package and install that * python setup.py bdist_debian * dpkg -i ienabler*.deb
